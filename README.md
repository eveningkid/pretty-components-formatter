# Formatter for Pretty Components

- computeStyle(name, style)
- extend(...args)
- fromCSSStringtoObject(string)
- fromObjectToCSS(style)
- parsePSS(string) for parsing `.pss` files
- stylesheet(identifier, style)
- util, *fromCamelToDash*
